# Componente `<search>`
  
## Propriedades
- *String **entityType*** - Tipo da entidade
- *String **pageTitle*** - Título da página
- *String **createEntityButton*** - Botão de criação da entidade

## Slots
- **default** - Para o uso das tabs
- **filter** - Para o uso dos filtros

### Importando componente
```PHP
<?php 
$this->import('search');
?>
```
### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<search></search>
```
