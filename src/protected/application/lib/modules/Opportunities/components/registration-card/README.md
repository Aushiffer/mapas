# Componente `<registration-card>`
Cards da tela 'minhas inscrições'
  
## Propriedades
- *Entity **entity*** - Entidade

### Importando componente
```PHP
<?php 
$this->import('registration-card');
?>
```
### Exemplos de uso
```HTML
<!-- utilizaçao básica -->
<registration-card :entity="registration"></registration-card>

```
